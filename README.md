
# Node APi with Prisma and JWT


## Run Locally

Clone the project

```bash
  git clone https://github.com/pavanvattikala/api-node
```

Go to the project directory

```bash
  cd api-demo
```

Install dependencies

```bash
  npm install
```

Setup Database and Seed

```bash
  npm run migrate
```

Start the server

```bash
  npm run start
```
